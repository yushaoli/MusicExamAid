# MusicExamAid PRD (working draft)

## Product goal
Build a WeChat Mini Program workflow that helps families and teachers complete music exam registration, review, lock, and export-ready submission data, while preserving exam records/certificates in a learner archive.

## Primary users
- Parent / guardian (account owner)
- Student (practice user)
- Teacher (review + confirmation)
- Institution admin (batch/export oversight)

## MVP scope
1. Learner profile management
2. Exam registration record creation per cycle
3. Teacher review + status transitions (draft/submitted/needs-change/confirmed/locked)
4. Archive of exam results/certificate metadata + attachments
5. Export mapping compatible with institution workflow

## Non-goals (MVP)
- Payments
- Automated scraping from official exam portals
- Full LMS/course management

## Success criteria
- Parent can submit a complete registration in <10 minutes
- Teacher can review and lock records from one queue
- Export output passes institutional import checks
- Archive records are retrievable by learner + cycle
